## 商品管理システム

## 概要
このシステムでは、商品の在庫管理を行うことができます。
商品登録、編集、削除を行うことができ、商品情報を随時更新することができます。

## 主な機能
・ログイン、ログアウト機能
・商品一覧画面
・商品新規登録、編集、削除機能
・商品検索機能
・会員情報編集、削除機能

## 開発環境
PHP 8.2.0
MySQL 5.7.34
Laravel 8.83.27
